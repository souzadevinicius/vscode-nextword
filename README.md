# Mocword README

Mocword English completion.

## Features

![screenshot](images/screenshot.gif)

- Display correct spellings.
Predict the next English words from the input sentence.

## Requirements

[**IMPORTANT**] You need to install [mocword](https://github.com/high-moctane/mocword) and
[mocword-data](https://github.com/high-moctane/mocword-data) in advance.

Please make sure that the mocword command can be used beforehand.
